# 安装 Fika

## 安装 Fika 之前

**始终**记住，Fika 是一个非官方的[SPT](https://sp-tarkov.com/#download)模组。在尝试安装 Fika 之前，您**必须**先安装一个可以正常工作的 SPT。在确保 SPT 可以正常工作之前，尝试安装 Fika 是毫无意义的。

这意味着您必须能够在 SPT 中顺利开始游戏，并且在安装 Fika 之前不应运行任何模组。请确保情况确实如此，然后再继续执行后续步骤。

如果您不确定如何安装 SPT，请按照[此处](https://hub.sp-tarkov.com/files/file/672-spt-installer/)的说明进行操作，完成后再返回此处。再次确保您的 SPT 安装正常，然后再继续。

我们建议您阅读[常规信息](https://project-fika.gitbook.io/wiki/general-information)部分，以便您更好地了解 Fika 的工作原理。

## 先决条件

- 您必须拥有最新且可以正常运行的 SPT 安装，**且未安装任何模块**。
    
- Fika_**不**_支持SPT 的最新版本。
    
- 确定您的 SPT 安装文件夹。这将被称为`SPT folder`。

# 安装FIKA专属客户

专用客户端是 Fika 独有的功能，可让您在单独的 Escape From Tarkov 实例上组织突袭。您可以卸载 AI 计算和其他资源密集型任务，以提高游戏性能。FPS 增益通常约为 25% 到 50%，具体取决于您的计算机规格和游戏中的机器人数量。

请注意，实现此目标需要一些技术知识。如果你完全没有经验，那将会很困难。我们的 Discord 会在这种情况下提供支持。

### 一般信息

- 专用客户端基本上是另一个会自动发起突袭的 Fika 实例。所有玩家都成为客户端（包括主机）。
    
- 为使专用客户端尽可能轻量，图形渲染被禁用。但是，专用客户端**不是**真正的专用无头服务器。它仍然需要所有游戏文件和大量 RAM 才能工作（尤其是对于街道等地图）。
    
- 换句话说，这是一个经过修补的《逃离塔科夫》实例，它拙劣地模仿了专用服务器。它的好处仍然很大，如果您有资源来改善您的游戏体验，我们建议您使用它。
    
- 建议在单独的机器上运行专用客户端。我们不建议使用 VPS；如上所述，这不是真正的专用无头服务器，并且运行专用客户端所需的资源对于 VPS 来说非常巨大。
    

### 硬件要求

- 现代 CPU 至少具有 4 个内核 @ 3.2GHz +。
    
- 16GB RAM（虚拟分页可以减轻这一要求，但会降低性能）。
    
- 带有 SSD/NVME 驱动器的 50GB 磁盘空间。不建议以任何形式将 HDD 用于 Fika。
    
- 不需要显卡。
    

### 安装

- 使用官方 BSG 启动器在您计划托管专用客户端的服务器上安装《逃离塔科夫》。这是验证您拥有该游戏的必要条件。
    
- 将您工作的 Fika 安装复制到服务器上的单独文件夹中。这将在以下步骤中引用`dedicated folder`。如果这不可能，请按照服务器上[安装 Fika 的](https://project-fika.gitbook.io/wiki/installing-fika)步骤进行操作，并确保`SPT.Launcher.exe`至少使用一次运行游戏。
    
- `<dedicated folder>\BepInEx\plugins`删除除`Fika.Core.dll`和子文件夹之外的所有 DLL `spt`。
    
- 下载[Fika.Dedicated](https://github.com/project-fika/Fika-Dedicated/releases/latest)
    
- 提取`Fika.Dedicated`至您的`dedicated folder`。
    
- 转到托管 SPT 服务器的机器。
    
- 导航到`<SPT folder>\user\mods\fika-server\assets\configs`并打开`fika.jsonc`。
    
- 查找该`dedicated`部分。
    
- 将`amount`配置文件的数量至少设置为`1`。这将在下次 SPT 服务器启动时生成专用的客户端配置文件。
    
- 将 设置`forceIp`为 SPT 服务器的 IP。这将告诉专用客户端要连接到哪个 IP。
    
- 保存并关闭`fika.jsonc`。
    
- 开始`SPT.Server.exe`。
    
- 您应该在控制台中看到以下输出：
```
ModLoader: loading: 1 server mods...
Mod: server version: 2.2.8 by: Fika loaded
Server: executing startup callbacks...
Importing database...
Database import finished
Found 0 dedicated client profiles.
Created 1 dedicated client profiles!
Generated launch script: /fika-server/assets/scripts/Start_dedicated_674e72ab0001917ea42a90fa.bat
Started webserver at http://0.0.0.0:6969
Started websocket at ws://0.0.0.0:6969
Server is running, do not close while playing SPT, Happy playing!!
```

- 导航到`<SPT folder>\user\mods\fika-server\assets\scripts`批处理文件并复制它。
    
- 将批处理文件粘贴到您的 中`dedicated folder`。它必须与 位于同一目录中`EscapeFromTarkov.exe`。
    
- 运行批处理文件以启动专用客户端。应出现一个控制台窗口。**不要关闭它**。
    
- `SPT.Launcher.exe`返回到您的常规 Fika 文件夹并像平常一样开始游戏。
    
- 进入大厅屏幕并点击`Host Raid`。勾选`Use dedicated host`复选框以使用专用主机开始突袭。
    

**警告**

专用客户端不应运行除 AI 或生成模块（如 SAIN、MOAR lite 或 SWAG+DONUTS）之外的任何模块。它们之所以必要，只是因为突袭主机处理 AI 和生成。

运行专为游戏体验而设计的模组将会导致专用客户端出现问题。一些值得注意的例子：Amanda.Graphics、MoreCheckmarks、EFTApi、GamePanelHud、DynamicMaps、LootValue、Ram Cleaner Interval、DeClutter 等。

如果您在启动专用客户端时遇到任何问题，这很可能是原因。

### 附加信息

- 如果`Use dedicated host`复选框呈灰色，则表示专用客户端无法连接到您的 SPT 服务器或专用客户端已在托管突袭。
    
- 以与解决普通 Fika 实例相同的方式解决连接问题。

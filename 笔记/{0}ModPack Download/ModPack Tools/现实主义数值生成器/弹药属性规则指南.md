# Realism 弹药数值规则指南

本文档用于说明《现实主义 (Realism)》模组中弹药（Ammo）补丁的自动生成规则。

规则实现文件：`ammo_rule_ranges.py`
规则应用入口：`generate_realism_patch.py` 中 `apply_realism_sanity_check()` 的 `RealismMod.Ammo` 分支。

---

## 1. 弹药物品（Ammo）必填基础字段

所有弹药物品数值对象建议至少包含以下基础属性：

- "$type": "RealismMod.Ammo, RealismMod"
- "ItemID": 物品 MongoID
- "Name": 内部引用名称
- "LoyaltyLevel": 商店解锁等级（1-4）
- "BasePriceModifier": 基础价格系数

---

## 2. 本规则覆盖字段

当前按“口径 + 穿深”分档重算以下 12 个字段：

- "InitialSpeed"
- "BulletMassGram"
- "Damage"
- "PenetrationPower"
- "ammoRec"
- "ammoAccr"
- "ArmorDamage"
- "HeatFactor"
- "HeavyBleedingDelta"
- "LightBleedingDelta"
- "DurabilityBurnModificator"
- "BallisticCoeficient"

说明：
- 第一层：按口径档位提供基础区间。
- 第二层：按穿深档位对第一层区间做增量修正（delta）。
- 最终值会在修正后的区间内进行加权采样，以保留原始输入风格并约束异常值。

---

## 3. 口径档位（第一层）

以下档位对应 `ammo_rule_ranges.py` 的 `AMMO_PROFILE_RANGES`：

### 3.0 EFT 手感子档（v2）

本次新增的“子档”会优先于大类匹配，用于强化 EFT 原版常见口径差异：

- `rifle_545x39`：低后坐、较高初速，偏稳定连发
- `rifle_556x45`：高初速、平衡后坐与穿深
- `rifle_762x39`：较高肉伤、后坐更明显
- `rifle_762x51`：全威力步枪弹，后坐与穿甲压力更高
- `rifle_9x39`：低初速重弹头，亚音速重击风格
- `rifle_300blk`：超/亚音速混合风格，参数跨度更大

子档关键词示例（完整见 `AMMO_PROFILE_KEYWORDS`）：

- `5.45x39` -> `rifle_545x39`
- `5.56x45 / .223` -> `rifle_556x45`
- `7.62x39` -> `rifle_762x39`
- `7.62x51 / .308` -> `rifle_762x51`
- `9x39` -> `rifle_9x39`
- `.300 AAC / 300 Blackout / .300 BLK` -> `rifle_300blk`

注：这些子档与后续穿深分层（第二层）叠加生效。

## 4. 穿深档位（第二层增量修正）

以下档位对应 `AMMO_PENETRATION_TIERS` 与 `AMMO_PENETRATION_MODIFIERS`：

- `pen_very_low`: 0 到 19
- `pen_low`: 20 到 29
- `pen_medium`: 30 到 39
- `pen_high`: 40 到 49
- `pen_very_high`: 50 到 64
- `pen_ap_extreme`: 65+

增量修正规则说明：

- 低穿深档位：通常提高伤害/流血倾向，降低穿甲与护甲伤害。
- 中穿深档位：保持平衡，接近口径基础区间。
- 高穿深档位：通常提高穿深/护甲伤害与热负担，降低肉伤与流血倾向。

注：最终区间 = 口径基础区间 + 穿深增量区间（按字段分别相加）。

---

## 5. 口径与穿深识别来源

脚本会从以下字段综合识别：

- 口径：`Caliber / AmmoCaliber / ammoCaliber / ammoType / Name`
- 穿深：`PenetrationPower`（若缺失则尝试 `Penetration / penPower`）

当口径无法识别时，默认使用 `intermediate_rifle`；
当穿深无法识别时，默认使用 `pen_medium`。

---

## 6. 调参建议

- 想让某口径整体更强/更弱：改 `AMMO_PROFILE_RANGES` 对应档位。
- 想调“同口径内 AP/HP 差异”：改 `AMMO_PENETRATION_MODIFIERS`。
- 想补充新口径识别：改 `AMMO_PROFILE_KEYWORDS`。
- 想改变穿深分层阈值：改 `AMMO_PENETRATION_TIERS`。
